# CAD Hydrofoil

Semester project focused on developing a self-stabilizing RC hydrofoil.

The system uses a T-foil design with custom PCB-based water sensors acting as the control “legs” to detect water contact and stabilize the craft in real time.

Core components:
- Custom-designed hydrofoil (CAD)
- Self-manufactured PCBs (water sensing + control)
- Arduino-based control system

The goal is to create a fully integrated, self-stabilizing hydrofoil platform combining mechanical design, electronics, and embedded control.
